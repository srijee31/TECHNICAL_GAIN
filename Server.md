# 🖥️ What is a Server?

## 1. Introduction

A **server** is a computer or system that provides data, resources, or services to other computers over a network.

The computers that request these services are called **clients**.

Example:

Your phone or laptop → Client  
Computer providing data → Server

---

## 2. Real-Life Example

Think of a **restaurant**.

Customer → orders food  
Kitchen → prepares and sends food

In this case:

Client → sends request  
Server → processes request and sends response

---

## 3. Example in Real Life

When you open **YouTube**:

1. Your phone sends a request
2. The request goes to YouTube's servers
3. The server sends the video data back
4. The video plays on your screen

---

## 4. Types of Servers

| Server Type | Purpose |
|-------------|--------|
| Web Server | Hosts websites |
| Database Server | Stores and manages data |
| File Server | Stores files |
| Mail Server | Handles emails |

---

## 5. Where Servers Are Located

Servers are usually stored in large facilities called **data centers**.

These centers contain thousands of powerful computers connected to the internet.

Example companies with massive data centers:

- Google
- Amazon
- Microsoft

---

## 6. Simple Summary

A **server** is a computer that receives requests from other devices and sends back the required data or service.
