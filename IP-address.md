# 🌍 What is an IP Address?

## 1. Introduction

An **IP Address (Internet Protocol Address)** is a unique number assigned to every device connected to the internet.

It allows devices to **identify and communicate with each other** on a network.

Without IP addresses, devices would not know **where to send or receive data**.

---

## 2. Real-Life Analogy

An IP address is like a **home address**.

Example:

If someone wants to send you a letter, they need your home address.

Similarly, when data travels on the internet, it needs the **IP address of the destination device**.

---

## 3. Example of an IP Address
192.168.1.1


This is a typical **IPv4 address**.

---

## 4. Types of IP Addresses

### IPv4

IPv4 uses **32-bit numbers**.

Example:
192.168.0.1


There are about **4.3 billion IPv4 addresses** available.

---

### IPv6

Because the internet is growing rapidly, IPv6 was introduced.

Example:
2001:0db8:85a3:0000:0000:8a2e:0370:7334


IPv6 provides **trillions of possible addresses**.

---

## 5. Public vs Private IP

| Type | Description |
|------|-------------|
| Public IP | Used on the internet and visible globally |
| Private IP | Used inside local networks like home WiFi |

Example:

Your router may assign your phone a private IP like:

192.168.1.5


---

## 6. Why IP Addresses Are Important

IP addresses help to:

- Identify devices on a network
- Route internet traffic
- Enable communication between devices
- Deliver data to the correct destination

---

## 7. Simple Summary

An **IP address** is a unique identifier for devices on the internet that allows them to send and receive data across networks.
