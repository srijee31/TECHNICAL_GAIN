# What is CORS?

## Definition

CORS (Cross-Origin Resource Sharing) is a security feature used by browsers.

It controls how resources can be requested from another domain.

---

## Simple Meaning

It decides:

👉 "Can this website access data from another website?"

---

## Example

Frontend: http://example.com  
Backend: http://api.com  

If frontend tries to call backend:

👉 Browser checks CORS rules

---

## Why CORS?

To prevent unauthorized access and improve security.

---

## Key Points

- CORS is enforced by browsers
- It protects users from malicious websites
- Controlled using headers
